## Quick Setup

### Prerequisites
```bash
node --version  # Node.js 16+
npm --version
```

### Create React App
```bash
npx create-react-app my-app
cd my-app
npm start
```

### With Vite (Faster)
```bash
npm create vite@latest my-app -- --template react
cd my-app
npm install
npm run dev
```
