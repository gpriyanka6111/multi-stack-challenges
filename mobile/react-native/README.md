## Quick Setup

### Prerequisites
```bash
node --version  # Node.js 16+
npm --version
# For iOS: Xcode (macOS only)
# For Android: Android Studio
```

### Create React Native App
```bash
npx react-native init MyApp
cd MyApp
# iOS
npx react-native run-ios
# Android
npx react-native run-android
```
