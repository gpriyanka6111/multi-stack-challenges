## Quick Setup

### Prerequisites
```bash
kubectl version  # Kubernetes 1.24+
# Minikube for local development
minikube version
```

### Running Projects
```bash
# Start Minikube
minikube start
# Apply configurations
kubectl apply -f deployment.yaml
# Check status
kubectl get pods
```
