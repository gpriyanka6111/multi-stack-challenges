## Quick Setup

### Prerequisites
```bash
docker --version  # Docker 20.10+
docker-compose --version
```

### Running Projects
```bash
# Build image
docker build -t image-name .
# Run container
docker run -p 8080:8080 image-name
# Docker Compose
docker-compose up -d
```
