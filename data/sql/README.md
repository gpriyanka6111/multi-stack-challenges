## Quick Setup

### Prerequisites
```bash
# MySQL
mysql --version
# Or SQLite
sqlite3 --version
```

### Running Queries
```bash
# MySQL
mysql -u username -p database_name < script.sql
# SQLite
sqlite3 database.db < script.sql
```
